Platform Abstraction Layer

The platform abstraction Layer contains the wrapper functions which provide the seamless interface between the MAC/PHY software to PAL modules

PAL provide interface for following components

  1. 1.PALTimer services for software timers and delay routines
  2. 2.Access to persistent storage(for example Flash, NVM or EEPROM)
  3. 3.NVIC module for enabling/disabling the RF and Arbiter interrupts
