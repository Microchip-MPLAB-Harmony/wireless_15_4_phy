Transceiver Feature Access

PHY Layer provides the access to following features of Transceiver

  1. 1.Configurable PSDU datarate - 250 Kbps, 500 Kbps, 1Mbps, 2Mbps
  2. 2. Tx Outpower up to +12 dBm
  3. 3.Hardware assisted MAC - Auto retry, auto acknowledgment, channel access back-off
  4. 4.Programmable Rx mode
    • -103 dBm - Continuous Rx and Promiscuous Mode
    • -98 dBm - RPC (Reduced Power consumption Mode
  5. 5.Continuous Transmission support - For certification usecases
  6. 6.Error detection - CRC-16 computation
  7. 7.Antenna Diversity
