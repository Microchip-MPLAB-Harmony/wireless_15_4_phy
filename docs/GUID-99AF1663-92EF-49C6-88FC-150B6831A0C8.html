PHY Data Structures

Following are the data structures used by the PHY Layer

Table 1. PHY Data Structures
Name Type Description
PHY_ConfigParam_t Enum PHY Configuration parameters supported by transceiver
PHY_FrameInfo_t Struct PHY_FrameInfo_t holds the data to be transmitted or the data being received by the transceiver
PHY_SleepMode_t Enum Sleep Modes supported by transceiver
PHY_CSMAMode_t Enum List of carrier sense multiple access with collision avoidance supported by PHY Layer
PHY_ContinuousTxMode_t Enum List of Continuous Transmission Test Modes supported by transceiver
PHY_Retval_t Enum List of return status for the PHY functions
PHY_TrxState_t Enum Enumeration for Transceiver States that can be set |
PHY_TrxStatus_t Enum Enumeration for current state of the Transceiver
